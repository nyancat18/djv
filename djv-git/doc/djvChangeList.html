Version 1.0.5

Features:

  • Always opening djv window in a certain size (feature request #42)
  • Add preferences for changing the font (feature request #41)
  • More detailed error messages and logging (feature request #40)
  • Linux: Move .sh-launchers to /usr/bin (feature request #38)
  • .desktop - wrong path to binary (feature request #36)

Bug fixes:

  • Errors opening QuickTime files on Windows (bug #28)
  • Default Playback speed isn't used (bug #27)
  • Add support for negative frame numbers (bug #26)
  • Linux build does not start (bug #25)

Version 1.0.4

Features:

  • DJV always offset/off screen (feature request #24)
  • Add support for Qt 4.8 (feature request #23)
  • Add time/frame ticks to the djv_view timeline (feature request #21)

Bug fixes:

  • Improve file browser performance (bug #24)
  • Allow large contrast/brightness stretching (bug #23)
  • Make the djv_view file browser non-modal again (bug #18)

Version 1.0.3

Features:

  • Add file type associations on OS X (feature request #22)
  • Add time/frame ticks to the djv_view timeline (feature request #21)
  • Add file type associations on Linux/Gnome3 (feature request #18)
  • Add file type associations on Windows (feature request #17)

Bug fixes:

  • File names with very large numbers are causing a crash (bug #20)
  • Hide controls when going full screen (bug #19)
  • Shortcuts not working for Window Show Bars (bug #17)

Version 1.0.2

Features:

  • Add an action to toggle playback in djv_view (feature request #16)
  • Add an action for reloading the current frame in djv_view (feature request #12)

Bug fixes:

  • The djv_view icon has an aliased appearance on Windows (bug #14)
  • Don't show terminal when launching djv_view on Windows (bug #13)
  • Add additional error handling to check whether images are compatible with the version of OpenGL being used (bug #10)
  • Release image file handles when the file cache is enabled (bug #9)
  • Add additional image scales to djv_view (bug #8)
